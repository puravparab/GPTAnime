## Installation

1. Clone the repository
	```
	git clone git@github.com:puravparab/GPTAnime.git
	```

2. Change directory
	```
	cd GPTAnime/src
	```

3. Install dependencies
	```
	bun install
	```

4. Run server
	```
	bun run dev
	```
